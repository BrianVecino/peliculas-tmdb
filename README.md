⚙️ Instalación y uso

Cloná el repositorio:

bash   git clone https://github.com/BrianVecino/peliculas-tmdb.git

Obtené una API Key gratuita en TMDb
En main.js, reemplazá TU_API_KEY con tu clave:

js   const API_KEY = 'TU_API_KEY';

Abrí index.html en tu navegador

✨ Funcionalidades

Listado del Top 20 de películas mejor valoradas
Filtro por género mediante un menú desplegable
Consumo de datos en tiempo real desde la API de TMDb

🔑 API
Este proyecto utiliza los siguientes endpoints de TMDb:

GET /movie/top_rated — Top 20 películas mejor valoradas
GET /genre/movie/list — Lista de géneros disponibles

Documentación oficial: developers.themoviedb.org
